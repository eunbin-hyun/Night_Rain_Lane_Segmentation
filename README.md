## ⚙️ Tech Stack

| 🚀**Hardware** | 💻**Software** |
|--------------|--------------|
| **Raspberry Pi 5** – 실시간 영상 처리 및 AI 모델 추론 | **OpenCV** – 이미지 전처리(CLAHE), 밝기조절 |
| **Picamera v3** – YOLO Segmentation 입력 영상 수집 | **YOLOv11 Segmentation** – 실시간 차선·객체 인식 |
| **Polarizing Film** – 야간·우천 시 반사광 억제 |
